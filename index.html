HTML

  • The head element contains information, or metadata about the webpage.
  • The body element represents the visible content shown to the user.
  • "<h#>" is code that represents a header and is followed by a number designating which tier of headings it is.
  • "<ul>" is code that represents an unordered list of items.
  • "<li>" is code that represents a bullet point on the list.
  • "&,l,t, and ; will print left facing angle bracket.
  • "&,g,t, and ; will print right facing angle bracket
  • < and > are considered html entities.
  • Html codes for entities

CSS

  • CSS is used to style a webpage. Color and size of headings and links can be changed using CSS.
  • It can be used to create a layout or for effects like adding animation.
  • a margin indicates how much space we want around the outside of an element.
  • a padding indicates how much space we want around the content inside an element.
  • { is code that represents the start to CSS.
  • } is code that represents the end to the CSS
  • &,#,1,2, and 3 will print a left curly bracket in html.
  • &,#,1,2, and 5 will print a right curly bracket in html.

Git

  • git status: checks what branch we're currently on.
  • git checkout main: allows you to switch to the main branch.
  • git checkout -b "branch name": creates a new branch and switches to it.
  • git pull: updates your local branch.
  • git add -A: stages files "lets git know to record changes to files"
  • git commit -m "changes listed": commits changes and describes the purpose of the commit.
